Process - CSS ZenGarden Brutalism

Voor mijn CSS ZenGarden-uitdaging heb ik een website gemaakt als onderdeel van de CSS ZenGarden-uitdaging. Dit is een individueel project waarbij we de html-code van een website nemen en css gebruiken om deze volledig te veranderen. Geen HTML, JAVASCRIPT of PHP. Mijn thema was Brutalisme, een stijl die ik voor de website moest gebruiken. Het is simplistisch van vorm met voornamelijk grijstinten en een zeer strikt thema met veel nadruk op sterke lettertypes.

In eerste instantie begreep ik de vereisten voor dit project verkeerd, omdat er twee verschillende vormen van Brutalisme zijn, als architectuurstijl en als website-stijl. Na wat verduidelijkingen was ik op het juiste pad. Ik begon met wat onderzoek naar de sfeer van de website. Wat is Brutalisme? Hoe kan ik hier een stijl van maken? Welke kleuren kan ik gebruiken? Na het ontwikkelen van een moodboard en de eerste ontwerpstappen kon ik beginnen met schetsen, evoluerend van een lage naar een hoge mate van detail. Uiteindelijk slaagde ik erin om de volledige visie van mijn prototype in code te vertalen. De structuur en de hele webpagina waren voltooid. Het was een uitdagend project, maar ik ben tevreden met het resultaat.
